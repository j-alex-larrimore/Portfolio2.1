Technologies

  • HyperText Markup Language (HTML)

    HTML is the standard language used to create web pages. HTML makes it simple for web designers to include text, links, images and input fields on their page.

  • Cascading Style Sheets (CSS)

    CSS is a tool for stylizing web pages. CSS separates content from design elements which makes it easier to change style without altering content or add content without altering style.

  • Bootstrap

    Bootstrap is a system of templates and tools that allow a user to easily create and stylize responsive web pages.
